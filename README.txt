"AI" Name: Mia

The point of this project is to create a personal "AI" that knows what I like and can provide things
based around this information without the need of any kind of user interaction.

An example case would be:
I hardcode what I like into the "AI", ex: Music by the rap artist Bugzy malone
Step 1: Bugzy Malone releases a song
Step 2: The "AI" detects this and uses this information to tell me that the artist has released a new song.
Step 3: Profit.
